# Teorema de Pitágoras - Página Web Interactiva

Página web educativa sobre el Teorema de Pitágoras con simulador, calculadora y quiz.

## ¿Qué incluye?
- Explicación visual de a² + b² = c²
- Calculadora de triángulos rectángulos
- Simulador arrastrable
- Quiz interactivo
- Ejemplos de la vida real
